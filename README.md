# Solidity base64 and base64URL enconder/decoder
This is an implementation of base64/base64URL, supporting Solidity **0.8.x**

## Set up
Node >= 10.x && yarn > 1.x
```
$ node --version
v16.15.0

$ npm install --global yarn

$ yarn --version
1.22.18
```

Install dependencies
```
$ yarn
```
## Test
1. Compile contracts
```
$ yarn compile
```
2. Run unit tests
```
$ yarn test
```
